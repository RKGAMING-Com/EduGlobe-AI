# EduGlobe AI

EduGlobe AI is a student-first learning platform with an AI teacher experience, homework support, content generation, and study planning.

## Features
- Landing experience for first-time visitors
- Student, teacher, parent, and admin role-based views
- Secure auth flow with JWT-based sessions
- AI teacher panel and homework workflow
- Content studio and planner pages

## Run locally

### 1) Install dependencies

Frontend
```bash
cd frontend
npm install
```

Backend
```bash
cd backend
npm install
```

### 2) Start the backend
```bash
cd backend
node server.js
```

The API will run on http://localhost:5000

### 3) Start the frontend
```bash
cd frontend
npm run dev
```

The app will be available at http://localhost:5173

### 4) Try the app
- Open the frontend URL in your browser
- Register a new account or use the existing demo login credentials
- Explore the dashboard, homework, content, and planner pages
